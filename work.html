Description:
I created a full-featured cryptocurrency that allows for peer-to-peer transactions between any two parties, anywhere in the world, as long as they have an internet connection. No central server or central bank is required to send or receive currency, as all transactions are mediated by a node program that can run on any consumer PC. The node program turns any computer that it runs on into a node, which is one of a network of voluntarily-run computers that build a decentralised ledger to keep track of every transaction. This ledger is known as a blockchain, and is the data structure that all cryptocurrencies are built on.

I built the node program with Python, which I used to impement the blockchain validation mechanism, known as the proof-of-work. I connected each node to the wider internet with Flask to convert every node into a server that can maintain the blockchain from anywhere in the world with an internet connection. I then used JavaScript, HTML, and CSS to build a wallet client to store and send currency, and this wallet can be accessed via a web interface. I use IndexedDatabase to store wallet details, allowing users to create multiple wallets and secure them with encrypted passwords.
